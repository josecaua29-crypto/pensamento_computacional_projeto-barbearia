Sistema de Gerenciamento - Barbearia

Projeto desenvolvido em Python com o objetivo de auxiliar na organização e gestão de uma barbearia, controlando fila de clientes, atendimentos, serviços e faturamento diário.

📌 Descrição

O projeto_app_Barbearia.py é um sistema simples executado via terminal que simula o funcionamento básico de uma barbearia. Ele utiliza estruturas fundamentais da linguagem Python para gerenciar:

Entrada de clientes
Ordem de atendimento
Serviços prestados
Controle de preços
Relatórios do dia

Ideal para fins educacionais e prática de lógica de programação.

🚀 Funcionalidades
👥 Fila de Espera (FIFO)
Adiciona clientes conforme chegam
Respeita a ordem de atendimento (primeiro a entrar, primeiro a sair)
📅 Atendimento
Chama clientes da fila para atendimento
Registra o serviço realizado
💰 Tabela de Serviços
Serviços cadastrados com preços
Exemplos:
Corte
Barba
Degradê
Sobrancelha
📊 Relatório Diário
Mostra os serviços realizados
Identifica serviços únicos com uso de set
Exibe faturamento total
⚙️ Painel Administrativo
Permite alterar preços em tempo real
🧮 Calculadora Integrada
Realiza cálculos simples
Aplica descontos e acréscimos em porcentagem
🛠️ Tecnologias Utilizadas

Cliente entra na fila

Barbeiro chama o próximo cliente

Sistema registra e soma ao faturamento
